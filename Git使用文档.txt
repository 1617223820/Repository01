1.要提交的文件必须放在Git_resposity文件夹内，即工作区内

2.查看版本：git --version

3.清屏：clear

4.设置签名：
设置邮箱：git config --global user.name "***"
设置邮箱：git config --global user.email "***"

5.本地仓库初始化：在本地文件夹内点开git bash,输入 git init

6.查看文件信息：ll   &&(隐藏项目)ll -la

*7.文件提交到暂存区：git add 文件名

*8.将暂存区的内容提交到本地库：git commit -m "此填注释的内容" [文件名]

9.查看工作区和暂存区的状态：git status

10.文件内容更改也需要重新add和commit

11.查看历史记录，即查看日志：git log

12.日志下一页：空格  上一页：B  退出：Q  到尾页显示END

13.日志信息简化显示：
git log --pretty=oneline
git log --oneline
*git reflog

*14.前进或后退到历史版本：git reset --hard [想跳的版本的索引]，所有区域都会移动

15.本地库的指针移动的同时，重置暂存区，但是工作区不动：git reset  --mixed

16.本地库指针移动，暂存区，工作区都不移动：git resrt --soft

*17.删除文件：rm [文件名]（只是删除工作区的文件，还需要用add和commit操作同步到暂存区和本地库）

18.删除的文件依然可以回退

19.比较工作区和暂存区里面所有文件：git diff

20.比较工作区和暂存区里面的文件：git diff [需要比对的文件名]

21.比较暂存区和工作区中文件的差异：
git diff [历史版本索引] [文件名]
git diff HEAD [文件名]

*22.创建分支：git branch [创建的分支名]
查看分支：git branch -v (查看分支时，*在哪个分支前面就说明此时就在该分支上)
切换分支：git checkout [分支名]
合并分支：git merge [需要合并的分支名]  （合并后会一直处于合并状态，必须提交至本地库才能结束合并状态。并且在commit时不能在后面加文件名）

23.查看文件内容：cat [文件名]

24.给远程库起别名：git remote add [别名] [远程库的地址]

25.查看远程库是否有别名：git remote -v

*26.将本地库的内容推送到远程库：git push [远程库别名或远程库地址] [要推送的分支]

*27.远程库内容克隆(提取)操作：git clone [远程库地址]

28.邀请其他人加入团队：打开自己的远程库-->setting-->collaborators-->add people-->输入名称邀请

*29.拉取操作pull操作，相当于fetch+merge。作用是相当于项目经理将更新的远程库提取到本地。
git fetch [远程库别名或地址] [远程库上对应的分支]  (抓取操作，只是将远程库的内容下载到本地，但是工作区的文件未更新，工作区还是原先的内容)
git merge [远程库别名或地址]/[远程库对应的分支]  (合并前需要注意分支是否切换回来)
*git pull [远程库别名或地址] [远程库上对应的分支]   (直接拉取操作，比上面方式简洁)
git pull [远程库别名或地址] [远程库上对应的分支]  --allow-unrelated-histories    (允许不相关历史合并)

30.协同开发时冲突的解决办法（多人不能同时对远程库的同一文件进行操作）:进行拉取操作-->人为解决冲突-->提交文件

31.删除Git本地缓存：管理Windows凭据-->删除对应凭据

32.跨团队合作：得到远程库地址-->进行fork操作(其他团队成员进入账号复制远程库地址，点击fork)-->clone本地进行修改然后提交至远程库-->
进行pull request操作(其他团队成员)-->项目经理审核，审核无误进行 merge pull request操作

33.SSH免密登录：.ssh文件位置：c/Users/杨/.ssh-->打开复制id_rad.pub文件内容-->在github个人主页setting里面进行设置

34.idea集成Git注意事项：一般先拉取（pull）操作，再进行推送（push）操作比较不容易冲突

35.利用idea直接克隆项目：file-->new-->project from version control-->URL输入远程库地址

36.idea解决冲突：会进行提示merge-->点击merge进行其他操作
